# 主机层面加固总览

本章涵盖三级等保中主机安全计算环境的加固方案，遵循 GB/T 22239-2019《信息安全技术 网络安全等级保护基本要求》安全计算环境控制点。

## 覆盖范围

| 类别 | 内容 | 参考标准 |
|------|------|----------|
| 操作系统 | Windows Server 2022、Linux (RHEL 9/CentOS) | GB/T 22239-2019 + CIS Benchmarks |
| 数据库 | Oracle 19c、MySQL 8.0 | GB/T 22239-2019 + CIS Benchmarks |
| 应用系统 | Tomcat 9、Nginx | GB/T 22239-2019 + CIS Benchmarks |
| 管理制度 | 等保三级管理制度要求 | GB/T 22239-2019 管理部分 |

## 安全计算环境控制点映射

每项加固措施均标注对应的 GB/T 22239-2019 安全计算环境控制点：

| 控制类 | 控制点 | 编号 |
|--------|--------|------|
| 身份鉴别 | 主标识/鉴别信息/复杂/登录失败/远程 | 7.1.4.1 a)~f) |
| 访问控制 | 权限分离/最小权限/默认共享 | 7.1.4.2 a)~c) |
| 安全审计 | 审计策略/覆盖/保护 | 7.1.4.3 a)~c) |
| 入侵防范 | 最小安装/补丁/组件禁用 | 7.1.4.5 a)~b) |
| 恶意代码防范 | 防病毒/集中管理 | 7.1.4.6 a)~b) |
| 数据加密 | 传输加密/存储加密 | — |
| 资源控制 | 会话/资源限制 | 7.1.4.8 a)~b) |
